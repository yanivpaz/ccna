# Packet tracer commands

## Priveleged user - configure the switch 
```
enable
configure t
```
## See current configurations
```
show running-config
show startup-config
```

## Copy configuration
```
copy runnning-config startup-config
OR
copy run start
```


## Keyboard shortcuts
```
ctrl+c - stop
ctrl+a - beginning of the line
ctrl+shift+6 - stop and break 

```
